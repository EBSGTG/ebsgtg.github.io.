Лабораторна робота №2. Тема роботи: Створення форм в html та позиціонування елементів CSS. Мета роботи: познайомитися з html-тегами та атрибутами для створення форм, отримати навички створення форм реєстрації, входу та зворотного зв’язку, та позиціонування елементів за допомогою різних видів.1. Вивчити html-теги та атрибути з лекційного матеріалу 1.1. про форми: 1.2. про блочну модель та позиціонування: height, width, border, padding, margin, box-sizing; position, left, right, top, bottom; float, clear; z- index; display (block, inline, inline-block). Додатково: transform: rotate(90deg); /* поворот елементу на 90° по часовій стрілці */ 2. Створити сторінку з наступними формами, використовуючи всі набуті знання по тегами і атрибутам та властивостям CSS: 2.1. входу, 2.2. реєстрації, 2.3. зворотного зв’язку, 2.4. реалізувати розміщення елементів за допомогою різних типів позиціонування, 2.5. використати CSS для оформлення сторінки. 3.
